## Kubectl
### CRUD Commands
``` 
 kubectl create deployment <name>
 kubectl edit deployment <name>
 kubectl delete deployment <name>
```

### Status Commands
``` 
 kubectl get nodes | pod | service | replicaset | deployment
```


### Debug Commands
``` 
 kubectl logs <pod name>
```
